# Настройка Telegram бота для обратной связи

## 1. Создание Telegram бота

1. Найдите в Telegram **@BotFather**
2. Отправьте команду `/newbot`
3. Следуйте инструкциям:
   - Введите имя бота (например: "CRM LMS Feedback Bot")
   - Введите username бота (должен заканчиваться на `bot`, например: `crm_lms_feedback_bot`)
4. Сохраните полученный **токен** (выглядит как `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

## 2. Получение Chat ID

1. Найдите в Telegram **@userinfobot** или **@get_id_bot**
2. Отправьте любое сообщение боту
3. Скопируйте ваш **Chat ID** (цифровой ID)
4. **ИЛИ** создайте группу/канал и добавьте туда бота, затем получите Chat ID группы

## 3. Настройка переменных окружения

Добавьте в файл `.env`:

```bash
# Telegram Bot для обратной связи
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
TELEGRAM_CHAT_ID=987654321
```

## 4. Применение миграций

Выполните в терминале:
```bash
python manage.py migrate
```

## 5. Проверка работы

1. Перезапустите Django приложение
2. Откройте в браузере `/feedback/`
3. Отправьте тестовое сообщение
4. Проверьте, что сообщение пришло в Telegram

## Функционал

- **Отзывы**: пользователи могут оставлять отзывы о платформе
- **Идеи**: предложения по улучшению функционала
- **Баги**: сообщения об ошибках и проблемах
- **Автоматическая отправка** в Telegram при создании сообщения
- **Административная панель** для управления обратной связью

## Структура URL

- `/feedback/` - главная страница с выбором типа обратной связи
- `/feedback/review/` - форма отзыва
- `/feedback/idea/` - форма идеи
- `/feedback/bug/` - форма бага
- `/feedback/success/` - страница успешной отправки
