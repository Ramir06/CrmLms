# Инструкция по добавлению ссылок на онлайн урок и чат

## Что было сделано:

### 1. Добавлены поля в модель Course
- `online_lesson_link` - ссылка на онлайн урок (Zoom, Google Meet и т.д.)
- `chat_link` - ссылка на чат (Telegram, WhatsApp и т.д.)

### 2. Обновлена форма создания курса
- В админ-панели добавлены поля для ввода ссылок
- Поля имеют подсказки с примерами форматов
- Поля не обязательные для заполнения

### 3. Обновлен навбар ментора
- Добавлены кнопки "Зайти на урок" и "Открыть чат"
- Кнопки появляются только когда ментор находится на странице курса
- Стиль навбара приведен в соответствие с админским

### 4. Создана страница курса для ментора
- URL: `/lms/mentor/course/<course_id>/`
- Показывает информацию о курсе и студентов
- В навбаре отображаются кнопки для доступа к уроку и чату

## Как использовать:

### Для администратора:
1. При создании или редактировании курса в админ-панели появились новые поля
2. В поле "Ссылка на онлайн урок" укажите ссылку на Zoom/Google Meet
3. В поле "Ссылка на чат" укажите ссылку на Telegram/WhatsApp чат
4. Сохраните курс

### Для ментора:
1. Перейдите на страницу курса: `/lms/mentor/course/<ID_курса>/`
2. В навбаре появятся кнопки "Зайти на урок" и "Открыть чат"
3. Кнопки ведут на ссылки, указанные администратором
4. Кнопки открываются в новой вкладке

## Примечания:
- Поля для ссылок не обязательные
- Если ссылки не указаны, кнопки в навбаре не отображаются
- Ссылки открываются в новой вкладке для удобства
- Навбар ментора теперь имеет такой же стиль как у администратора, но без кнопок настроек и расписания
