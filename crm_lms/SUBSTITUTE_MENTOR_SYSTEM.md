# Система замены менторов

## Обзор

Система позволяет назначать временных заменяющих менторов для конкретных уроков с гибкими настройками прав доступа.

## Функциональность

### 1. Создание замены
- **Кто может создавать:** Администраторы и основные менторы курса
- **Где:** В расписании урока (кнопка "Заменить ментора")
- **Процесс:** Выбор заменяющего ментора из списка доступных, указание причины

### 2. Права доступа заменяющего ментора
- ✅ **Отмечать посещаемость** - только на дни замен
- ✅ **Создавать материалы** - для уроков замены
- ✅ **Просматривать оценки** - только чтение
- ✅ **Создавать задания** - для уроков замены
- ❌ **Оценивать задания** - только основной ментор

### 3. Кабинет заменяющего ментора
- **URL:** `/mentors/substitute/`
- **Отображение:** Курсы с пометкой "ЗАМЕНА" (красным)
- **Доступ:** Только подтвержденные замены

### 4. Процесс работы

#### Для основного ментора/админа:
1. Зайти в расписание курса
2. Нажать кнопку "Заменить ментора" у нужного урока
3. Выбрать заменяющего ментора
4. Указать причину (опционально)
5. Отправить на подтверждение

#### Для заменяющего ментора:
1. Получить уведомление о назначении замены
2. Подтвердить замену в личном кабинете
3. Зайти в `/mentors/substitute/`
4. Выбрать курс с пометкой "ЗАМЕНА"
5. Работать с уроками в рамках прав доступа

## Статусы замены

- **pending** - Ожидает подтверждения
- **confirmed** - Подтверждено (активна замена)
- **completed** - Завершено (урок проведен)
- **cancelled** - Отменено

## URL маршруты

### Управление заменами:
- `GET /lessons/{lesson_id}/substitute/` - Страница замены ментора
- `POST /lessons/{lesson_id}/substitute/create/` - Создание замены
- `POST /lessons/substitutions/{id}/confirm/` - Подтверждение замены
- `POST /lessons/substitutions/{id}/cancel/` - Отмена замены

### Кабинет ментора:
- `GET /mentors/substitute/` - Список замещаемых курсов
- `GET /mentors/substitute/{course_id}/` - Детали курса замены

## Модели данных

### MentorSubstitution
- `lesson` - Урок для замены
- `original_mentor` - Основной ментор
- `substitute_mentor` - Заменяющий ментор
- `status` - Статус замены
- `reason` - Причина замены
- `created_by` - Кто создал замену

### SubstituteAccess
- `substitution` - Связанная замена
- `can_mark_attendance` - Может отмечать посещаемость
- `can_create_materials` - Может создавать материалы
- `can_view_grades` - Может просматривать оценки
- `can_create_assignments` - Может создавать задания
- `can_grade_assignments` - Может оценивать задания

## Шаблоны

- `lessons/substitute_mentor.html` - Страница создания замены
- `mentors/substitute_courses.html` - Список курсов замены
- `mentors/substitute_course_detail.html` - Детали курса замены

## Важные моменты

1. **Автоматическая проверка доступности:** Система проверяет, что у заменяющего ментора нет других уроков в это время
2. **Гибкие права:** Можно настроить права доступа для каждой замены индивидуально
3. **История:** Все замены сохраняются в истории с указанием кто создал и когда
4. **Уведомления:** Заменяющий ментор получает уведомление о назначении

## Пример использования

1. Основной ментор заболел на 15.03.2026
2. Администратор заходит в расписание, находит урок 15.03
3. Нажимает "Заменить ментора", выбирает доступного ментора
4. Заменяющий ментор получает уведомление и подтверждает
5. В день урока заменяющий ментор отмечает посещаемость и проводит урок
6. После урока статус замены меняется на "completed"

## Безопасность

- Только авторизованные пользователи могут создавать замены
- Проверка прав доступа на каждом этапе
- Основной ментор не может удалить замены другого ментора
- Заменяющий ментор получает доступ только к конкретным урокам
