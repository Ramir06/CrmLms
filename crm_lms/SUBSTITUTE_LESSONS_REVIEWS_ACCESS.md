# Исправление доступа к урокам и отзывам + Генерация ссылок для заменяющих менторов

## Проблемы

1. **Ошибка 404 для уроков**: `http://127.0.0.1:8000/mentor/courses/26/lessons/`
2. **Ошибка 404 для отзывов**: `http://127.0.0.1:8000/mentor/courses/26/reviews/`
3. **Отсутствие функционала генерации ссылок на отзывы для уроков замены**

## Решение

### 1. Исправлена система уроков (`apps/lessons/views.py`)

**Добавлены импорты:**
```python
from django.core.exceptions import PermissionDenied
from apps.core.mixins_substitute import check_substitute_access
```

**Создана функция `get_mentor_course_with_substitute_access()`** - аналогична функции в лекциях

**Обновлены все view функции:**
- `lessons_calendar()` - добавлен флаг `is_substitute_mentor`
- `lesson_create()` - поддержка заменяющих менторов
- `lesson_edit()` - поддержка заменяющих менторов  
- `lesson_delete()` - поддержка заменяющих менторов

### 2. Исправлена система отзывов (`apps/reviews/views.py`)

**Добавлены импорты:**
```python
from django.core.exceptions import PermissionDenied
from apps.core.mixins_substitute import check_substitute_access
from apps.lessons.models_substitute import MentorSubstitution
```

**Создана функция `get_mentor_course_with_substitute_access()`**

**Обновлены все view функции:**
- `reviews_list()` - фильтрация уроков только для замен
- `generate_feedback_link()` - проверка прав на генерацию ссылок
- `feedback_detail()` - поддержка заменяющих менторов

### 3. Функционал генерации ссылок на отзывы

**Для заменяющих менторов:**
- Показываются только уроки, на которые они назначены
- Можно генерировать ссылки только для своих уроков замены
- Проверка через `can_mark_attendance()` для подтверждения прав

**Фильтрация уроков в отзывах:**
```python
# Для заменяющего ментора показываем только уроки, на которые он назначен
if course.mentor != request.user and request.user.role == 'mentor':
    substitute_lesson_ids = MentorSubstitution.objects.filter(
        substitute_mentor=request.user,
        status='confirmed',
        lesson__course=course
    ).values_list('lesson_id', flat=True)
    lessons = lessons.filter(id__in=substitute_lesson_ids)
```

**Проверка прав на генерацию ссылок:**
```python
# Для заменяющего ментора проверяем, что он назначен на этот урок
if course.mentor != request.user and request.user.role == 'mentor':
    if not can_mark_attendance(request.user, lesson_id):
        return JsonResponse({'error': 'Вы можете генерировать ссылки только для уроков, на которые назначены'}, status=403)
```

## Как это работает

### Доступ к урокам
1. Заменяющий ментор заходит в `/mentor/courses/{id}/lessons/`
2. Видит календарь с уроками курса
3. Может просматривать детали уроков
4. **Ограничение**: не может создавать/редактировать/удалять уроки (только просмотр)

### Доступ к отзывам
1. Заменяющий ментор заходит в `/mentor/courses/{id}/reviews/`
2. Видит список только тех уроков, на которые назначен
3. Может генерировать ссылки для сбора отзывов
4. Может просматривать отзывы по своим урокам

### Генерация ссылок на отзывы
1. В списке уроков нажать "Сгенерировать ссылку"
2. Система создает уникальную ссылку для сбора отзывов
3. Ссылка ведет на публичную форму отзыва
4. Студенты могут оставлять отзывы без регистрации
5. Отзывы привязываются к конкретному уроку

## Безопасность

- **Проверка прав доступа**: на каждом уровне проверяется, является ли пользователь заменяющим ментором
- **Фильтрация контента**: показываются только релевантные уроки
- **Временные ограничения**: доступ работает только в рамках времени урока
- **Права на действия**: генерация ссылок только для назначенных уроков

## Визуальные индикаторы

В шаблоны добавлен контекст `is_substitute_mentor` для:
- Показа уведомлений о режиме замены
- Скрытия кнопок создания/редактирования
- Изменения текста и сообщений

## URL структуры

- Уроки: `/mentor/courses/{course_id}/lessons/`
- Отзывы: `/mentor/courses/{course_id}/reviews/`
- Генерация ссылки: `POST /mentor/courses/{course_id}/reviews/{lesson_id}/generate/`
- Публичная форма: `/reviews/feedback/{token}/`
