# Настройка редактора футера

## Что было сделано:

1. **Филиалы с кодами 0084**: 
   - Добавлено поле `code` в модель Organization
   - Автоматическая генерация кода формата 0084, 0085, 0086 и т.д.
   - Код отображается в админке организаций

2. **Редактор футера с паролем**:
   - Создана модель FooterContent для управления контентом футера
   - Добавлен парольный доступ к редактору
   - Скрытая секция в настройках системы (раздел "Другие...")
   - Динамический футер на всех страницах

## Как использовать:

### 1. Установка пароля для футера:
```bash
python manage.py setup_footer_password ваш_пароль
```

### 2. Доступ к редактору футера:
- Зайдите в `/admin/settings/`
- В разделе "Другие..." нажмите "Редактор футера"
- Введите пароль
- Редактируйте контент футера

### 3. Создание филиалов:
- Зайдите в админку Django
- Создайте новую организацию
- Код филиала сгенерируется автоматически в формате 0084

### 4. Настройка контента футера:
- Заголовок футера
- Описание
- Контактная информация
- Социальные сети (JSON формат)
- Дополнительные ссылки (JSON формат)
- Текст копирайта

## Формат JSON для ссылок:

### Социальные сети:
```json
{
    "telegram": "https://t.me/channel",
    "instagram": "https://instagram.com/profile",
    "facebook": "https://facebook.com/page"
}
```

### Дополнительные ссылки:
```json
{
    "Политика конфиденциальности": "/privacy/",
    "Условия использования": "/terms/",
    "Поддержка": "/support/"
}
```

## Примечание:
- Футер будет отображаться на всех страницах после включения
- Пароль хранится в системных настройках
- Доступ к редактору футера есть только у пользователей с правами доступа
