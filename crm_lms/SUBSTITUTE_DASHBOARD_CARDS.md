# Карточки курсов замен на главной странице ментора

## Функциональность

Система автоматически показывает карточки курсов для замены на главной странице ментора в день проведения урока. После окончания урока карточка исчезает.

## Как это работает

### 1. Отображение карточек
- Карточки курсов замен появляются только в день проведения урока
- Карточки показываются выше основных курсов ментора
- Каждая карточка имеет визуальное оформление в желтых тонах для отличия от основных курсов

### 2. Фильтрация по времени
- **До начала урока**: карточка отображается с бейджем "ВРЕМЯ УРОКА"
- **Во время урока**: карточка отображается с бейджем "ИДЕТ СЕЙЧАС"
- **После окончания урока**: карточка скрывается

### 3. Визуальные индикаторы
- Желтая рамка и фон для карточек замен
- Пульсирующий бейдж "ЗАМЕНА"
- Информация об основном менторе курса
- Время и дата проведения урока

### 4. Статистика
- Общая карточка "Уроков сегодня" подсвечивается желтым при наличии замен
- Показывает общее количество уроков включая замены

2. **`templates/mentor/dashboard/index.html`**

### Логика работы:

```python
# Получаем активные замены на сегодня
today_substitutions = MentorSubstitution.objects.filter(
    substitute_mentor=user,
    status='confirmed',
    lesson__lesson_date=today
).select_related('lesson', 'lesson__course', 'original_mentor')

# Фильтруем по времени (урок не должен закончиться)
active_substitutions = []
for substitution in today_substitutions:
    lesson = substitution.lesson
    lesson_end_datetime = datetime.combine(lesson.lesson_date, lesson.end_time)
    
    if timezone.now() < timezone.make_aware(lesson_end_datetime):
        active_substitutions.append(substitution)
```

## Права доступа

При переходе в курс по ссылке из карточки замены, ментор получает полный доступ к функционалу курса на время проведения урока:

- Посещаемость (только для этого урока)
- Материалы урока
- Студенты курса
- Задания (если есть)

## Временные рамки

- **Появление карточки**: в 00:00 дня проведения урока
- **Исчезновение карточки**: сразу после окончания времени урока
- **Доступ к курсу**: до окончания времени урока

## Особенности

1. **Автоматическое управление**: не требует ручного включения/выключения
2. **Временный доступ**: доступ к курсу ограничен временем урока
3. **Визуальная индикация**: четкое отличие от основных курсов
4. **Информативность**: показывает основного ментора и время урока

## Пример использования

Ментор Иванов назначен заменой на курс "Python Basic" 19 марта с 14:00 до 16:00.

- **До 14:00**: на главной странице появляется карточка курса с бейджем "14:00"
- **С 14:00 до 16:00**: карточка показывает бейдж "ИДЕТ СЕЙЧАС"
- **После 16:00**: карточка исчезает, доступ к курсу закрывается

При клике на карточку ментор получает доступ ко всем функциям курса как основной ментор, но только в рамках этого урока.
