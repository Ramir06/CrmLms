# Руководство по модулю "Бухгалтерия"

## Обзор

Модуль "Бухгалтерия" предоставляет полноценную систему управления финансовыми операциями организации с поддержкой мультиорганизационной структуры.

## 🚀 Основные возможности

### 1. **Дашборд бухгалтерии**
- 📊 Визуальная статистика доходов/расходов за месяц
- 📈 График динамики за последние 30 дней
- 🏦 Балансы по всем счетам в реальном времени
- 📋 Последние транзакции
- 📅 Навигация по месяцам

### 2. **Управление транзакциями**
- ➕ Добавление ручных транзакций
- ✏️ Редактирование (только ручных операций)
- 🗑️ Удаление (с защитой авто-сгенерированных)
- 🔍 Расширенные фильтры и поиск
- 📄 Пагинация больших объемов данных

### 3. **Типы операций**
- 💰 **Приход** - доходы от студентов, продаж и т.д.
- 💸 **Расход** - зарплаты, аренда, реклама и т.д.
- 🔄 **Перевод** - перемещение средств между счетами

### 4. **Автоматическая генерация**
- 💳 **Оплата курса студентом** → автоматический приход
- 💸 **Выдача зарплаты** → автоматический расход
- 🔒 **Защита** от редактирования авто-операций

### 5. **Мультиорганизационность**
- 🏢 **Строгая изоляция** данных между организациями
- 👤 **Фильтрация** по организации пользователя
- 🔐 **Безопасность** - нет доступа к чужим данным

### 6. **Управление счетами**
- 🏦 Создание и редактирование счетов
- 💰 Автоматический расчет балансов
- 📊 Статистика по каждому счету

### 7. **Категории операций**
- 📁 Создание категорий доходов и расходов
- 🎨 Цветовая маркировка
- 📊 Статистика по категориям

### 8. **Экспорт и отчеты**
- 📊 Экспорт в Excel с фильтрами
- 📈 Статистика за периоды
- 📋 Детализация по категориям

## 🎯 Навигация

### Основные URL:
- `/admin/finance/` - Дашборд бухгалтерии
- `/admin/finance/transactions/` - Список транзакций
- `/admin/finance/transactions/create/` - Добавление транзакции
- `/admin/finance/accounts/` - Управление счетами
- `/admin/finance/categories/` - Управление категориями
- `/admin/finance/export/` - Экспорт в Excel

## 📋 Структура данных

### FinanceTransaction (Транзакции)
```python
type = 'income' | 'expense' | 'transfer'
amount = Decimal(12, 2)
account = ForeignKey(FinanceAccount)
category = ForeignKey(FinanceCategory)
description = TextField()
transaction_date = DateField()
auto_generated = BooleanField()
related_entity_type = 'payment' | 'salary_accrual'
related_entity_id = PositiveIntegerField()
organization = ForeignKey(Organization)
```

### FinanceAccount (Счета)
```python
name = CharField(max_length=100)
balance = Decimal(14, 2)
description = TextField()
organization = ForeignKey(Organization)
```

### FinanceCategory (Категории)
```python
name = CharField(max_length=100)
type = 'income' | 'expense'
color = CharField(max_length=7)  # HEX цвет
organization = ForeignKey(Organization)
```

## 🔄 Автоматическая интеграция

### Платежи студентов
При создании платежа со статусом `completed`:
```python
# Автоматически создается транзакция
FinanceTransaction.objects.create(
    type='income',
    category='Оплата курсов',
    amount=payment.amount,
    description=f"Оплата курса: {course.name} - {student.name}",
    auto_generated=True,
    related_entity_type='payment',
    related_entity_id=payment.id
)
```

### Зарплаты менторов
При начислении зарплаты со статусом `paid`:
```python
# Автоматически создается транзакция
FinanceTransaction.objects.create(
    type='expense',
    category='Зарплаты',
    amount=salary.amount,
    description=f"Зарплата: {mentor.name}",
    auto_generated=True,
    related_entity_type='salary_accrual',
    related_entity_id=salary.id
)
```

## 🔧 Настройка и инициализация

### 1. Инициализация бухгалтерии
```bash
python manage.py init_finance
```
Команда создаст:
- Базовые категории доходов и расходов
- Стандартные счета (Основной, Карта, Наличные)
- Привяжет к существующим организациям

### 2. Настройка категорий
При необходимости создайте кастомные категории:
```python
FinanceCategory.objects.create(
    name='Консультации',
    type='income',
    color='#10b981',
    organization=organization
)
```

### 3. Настройка счетов
```python
FinanceAccount.objects.create(
    name='PayPal',
    balance=0,
    description='PayPal счет',
    organization=organization
)
```

## 🔒 Безопасность и права доступа

### Права доступа
- 👑 **Admin Required** - только администраторы
- 🏢 **Организационная изоляция** - пользователи видят только свои данные
- 🔐 **Защита авто-операций** - нельзя редактировать/удалять

### Безопасные операции
- ✅ Ручные транзакции - можно редактировать/удалять
- ❌ Авто-сгенерированные - только просмотр
- 🔍 **Аудит** - все операции логируются
