# Отображение последнего входа в админке

## Описание

Добавлено отображение времени последнего входа для студентов и менторов в административной панели. Это помогает администраторам отслеживать активность пользователей.

## Реализация

### 1. Добавление поля в модель CustomUser

**Файл:** `apps/accounts/models.py`

```python
# Добавлено поле для отслеживания последнего входа
last_login_at = models.DateTimeField(null=True, blank=True, verbose_name='Последний вход')

# Добавлен метод для форматирования времени
def get_last_login_display(self):
    """Возвращает отформатированное время последнего входа"""
    if self.last_login_at:
        return self.last_login_at.strftime('%d.%m.%Y %H:%M')
    return 'Никогда'
```

### 2. Обновление аутентификационного бэкенда

**Файл:** `apps/core/auth_backends.py`

```python
# При успешном входе обновляем время
if user.check_password(password) and self.user_can_authenticate(user):
    # Успешный вход - сбрасываем счетчик неудачных попыток
    self._clear_failed_attempts(username)
    
    # Обновляем время последнего входа
    user.last_login_at = timezone.now()
    user.save(update_fields=['last_login_at'])
    
    return user
```

## Формат отображения

Время последнего входа отображается в формате:
- **Формат:** `DD.MM.YYYY HH:MM`
- **Пример:** `19.03.2026 14:30`
- **Если входа не было:** `Никогда`
